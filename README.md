# KVKK Case Study – Personal Data Protection

## Project Type
Academic Project – Information & Data Law

## Overview
This project analyzes a practical case where personal data continues to be processed after membership termination.

## Content
- Data controller application process
- KVKK Articles 7, 11, 13, 14 analysis
- Complaint process to the Data Protection Authority

## Outcome
Legal evaluation of data protection violations and application procedures.

## Note
This project was completed as part of an academic course.
